# fuzzy-men (Server)
Rocket server, with decoder

## Build instruction
Prepare system

`sudo apt update && sudo apt install gfortran libopenblas-dev`

`yay -S gcc-fortran libopenblas`

Docker build

`docker build -t fuzzymen .`
